06 — Ícones
Sistema de ícones SVG
Os ícones são arquivos .svg referenciados via <img> com filtros CSS para troca de cor. Não há font icon — o sistema usa filtros: --filter-to-white · --filter-to-cian · --filter-to-red · --filter-to-gray.
